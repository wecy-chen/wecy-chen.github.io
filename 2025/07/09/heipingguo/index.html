openCore

全部操作在 OC 内 config 文件

添加 nvram 启动项

  1. 找到 Misc — Boot–Show Picker 并勾选；

  2. 找到 Misc — Boot–HideAuxiliary 确保取消勾选

切换 SIP 状态

Misc – Security – AllowToggleSip 勾选保存

恢复模式 OC 引导界面按空格

ToggleSIP(Enabled) 开启显示
ToggleSIP(Disabled) 关闭

教程

DSDT

工具

SSDTTime-master

需要下载,配置 python 环境

macOS 下操作方法

  1. 使用军刀 hackintool 工具提取导出 DSDT 文件 (导出放在新建文件中,有很多文件,实际用到一个 DSDT.aml)
  2. 打开 SSDTTime 脚本 选择 1 拖拽放入 DSDT.aml 文件 回车 修复默认回车
  3. OC 下 config 在 APCI 的 patch 修补下参数(两个编辑器窗口) 和 导入 SSDT-HPET.aml 文件

macOS 系统抓取 10 分钟内 kernel log 的详细信息命令：

log show –predicate ‘process == “kernel”‘ –debug –last 10m > ~/Desktop/kernel.log

零刻 bios

零刻知识库

https://doc.bee-link.com.cn/books/gti13-ultra-bios/page/pcie

  1. 切换 PCIE 通道速度
  • 切换 Chipset
  • 选择 System Agent (SA)Cofiguration
  • 选择 PCI Express Configuration
  • 选择 PCI Express Root Port 2
  • 设置 PCIe Speed [Auto Gen3 Gen4 …]可选

按 F4 保存并退出

  1. 来电自启设置
  • 切换 Advanced
  • 选择 PCH-IO Configuration
  • 设置 State After G3 [S0 State]开启 [S5 State]关闭

按 F4 保存并退出

  1. 调整 CPU 功率
  • 切换 Advanced
  • 选择 Power & Performance
  • 选择 CPU-Power Management Control
  • 选择 Config Tdp Configurations
  • 设置 TDP Configurable 【54W】和【65W】可选
